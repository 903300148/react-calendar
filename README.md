# react-calendar-ui

react-calendar-ui,基于react的日历ui组件

###  Installation

> npm insatall  react-calendar-ui -save

###  Usage

```javascript
import React from 'react'
import Calendar from 'react-calendar-ui'

class Text extends React.Component {
  render() {
	  return (
          <Calendar></Calendar>
	  )
   }
}
export default Text
```
